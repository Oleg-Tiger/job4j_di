job4j_di
===========

## Описание
Данный проект создан с целью выполнения учебных заданий курса job4j, при прохождении раздела обучения
"DI. Внедрение зависимостей". В данном проекте подключена сборка через Maven.

## Требуемое окружение:
- JDK17
- Maven 3.8.1
